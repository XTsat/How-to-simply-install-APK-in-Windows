一般来说你启动任何 WSA 应用都会自动启动 WSA

不过有时启动很慢

可以将 WSA 服务设置为连续

使子系统始终运行在后台(推荐 16G 内存及以上使用)

![WSA 连续](/photo/WSA/WSA%20setting%20连续.png)

## 无法连接 WSA 怎么办

有时候外部应用无法连接到内部的 WSA

可以打开 WSA 设置 将开发人员模式切换一次设置

![WSA 开发模式](/photo/WSA/WSA%20setting%20开发人员模式.png)
